[![](https://badge.imagelayers.io/plugins/drone-gh-pages:latest.svg)](https://imagelayers.io/?images=plugins/drone-gh-pages:latest 'Get your own badge on imagelayers.io')

Drone plugin for publishing to GitHub Pages. For the usage information and a listing of the available options please take a look at [the docs](DOCS.md).

## Binary

Build the binary using `make`:

```
make deps build
```

### Example

```sh
./drone-gh-pages <<EOF

    "repo": {
        "clone_url": "git://github.com/drone/drone",
```
